Install Hexkeep
Before you tap: the download is about 450 MB and needs roughly 1 GB free while it unpacks. Free up space first if the phone is nearly full.

If it installs but won’t open

Restart the iPhone. Since iOS 18, a manually installed app needs a reboot before the system will trust its provisioning profile. This is expected, not a failure.

If it still refuses, go to Settings → General → VPN & Device Management, find the developer profile, and tap Trust.

Why not TestFlight?

Apple’s TestFlight backend is currently refusing to serve this app (“The requested app is not available or doesn’t exist”) even though the build, the tester, the availability, the price, the agreements and the beta contract all verify as correct. This Ad Hoc install bypasses TestFlight entirely: iOS downloads the app directly and trusts it because the provisioning profile inside it names this exact device.
